# SPA Micro Starter

Vue 3 SPA + Zig microservices. The shell serves the Vue bundle
(`public/`) and is the home for future auth flows. Each feature
runs as its own JSON service binary. Caddy fronts the stack on
:8000 and routes by path prefix.

## Layout

```
build.zig                       shell
build.zig.zon
app.yaml                        shell identity + port (no proxy config)
Caddyfile                       generated by `planctl deploy`
dev.sh                          one-command build + run
public/                         vite build output
src/
  main.zig                      shell entry
  ctx.zig

services/
  tasks/                        example JSON service
    build.zig
    build.zig.zon
    db.yaml                     planck/db storage tuning + wire port
    service.yaml                identity + route + wasm http port
    src/
      app.zig                   WASM entry
      dev.zig                   native entry
      ctx.zig
      routes.zig
      repo.zig                  planck.Model wrapper
      models/task.zig
      handlers/                 list / create / toggle / delete (JSON)

frontend/
  package.json
  vite.config.js                emits to ../public/, proxies /tasks to Caddy
  index.html
  src/
    main.js
    App.vue
    style.css
    router.js
    pages/Tasks.vue
    composables/api.js
```
